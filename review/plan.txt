Setup
1. Install the Spackle plugin
2. Visit a customer or product page
3. Run the initial sync
4. Follow the quickstart instructions here: https://docs.spackle.so/#quick-start


Test application integration
1. Visit https://spackle-example-flask.onrender.com (this can take a minute to start)
2. Enter your api key found in the Spackle Settings page
3. Enter a customer ID to fetch the feature state for that customer
